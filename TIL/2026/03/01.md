# 2026-03-01 TIL: SSH 설정 및 Git 기초

## 1. 오늘 해결한 문제
- **에러**: `Permission denied (publickey)` 에러로 git clone 실패.
- **원인**: GitHub에 내 컴퓨터의 SSH 공개키가 등록되지 않음.
- **해결**:
    - `ssh-keygen -t ed25519`로 키 생성.
    - `cat ~/.ssh/id_ed25519.pub` 복사 후 GitHub Settings에 등록.
    - `ssh -T git@github.com`으로 연결 확인.
    - git clone 주소 원하는폴더명 으로 깃 클론 성

## 2. 새로 배운 개념
- `ed25519`: 최신 보안 알고리즘, 짧고 강력함.
